# Description List

A Description List component ready to use in your UI.  
The Description List is UI/UX agnostic, allowing you to provide your preferred styles.

When you import the Description List component without providing any className or style, it will look as follows:

<Canvas of={DescriptionListStories.Unstyled} />

A more complex example with all the possible properties is:

```jsx
<DescriptionList
  className="myStyle"
  descListProps={{ style: { color: 'red' } }}
  detailClassName="myStyle"
  termClassName="myStyle"
>
  <Term className="myStyle" termProps={{ style: { color: 'red' } }}>
    Coffee
  </Term>
  <Detail className="myStyle" detailProps={{ style: { color: 'red' } }}>
    - black hot drink
  </Detail>
  <Term>Milk</Term>
  <Detail>- white cold drink</Detail>
</DescriptionList>
```
