# ToolTip

ToolTip component ready to use in your UI

ToolTip is UI/UX agnostic so you need to provide your style to have the look and feel you prefer.

## Usage

Use the ToolTip component when users need to show some infomration/tips.

An example with all the available properties is:

```js
<ToolTip
  content={content}
  direction="bottom"
  className="tooltipClass"
  background="red"
  color="white"
>
  <b> Testing Bottom </b>
</ToolTip>
```
