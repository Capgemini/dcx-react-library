# Styled Vertical DragResize

In this section we're using the vertical DragResize component with custom classes for the container, the two panels and the divider.

<Canvas>

# Styled Horizontal DragResize

In this section we're using the horizontal DragResize component with custom classes for the container, the two panels and the divider.

<Canvas>
  <Story name="styled-horizontal-divider">
    {() => {
      
      return (
        <DragResize
          orientation="horizontal"
          firstPane={<div>top panel content</div>}
          secondPane={<div>bottom panel content</div>}
          dividerClassName="custom-horizontal-divider"
          firstPaneClassName="first-pane"
          secondPaneClassName="second-pane"
          containerClassName="custom-container"

      );
    }}

</Canvas>
