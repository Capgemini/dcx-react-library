# DragResize

A DragResize component to split the view pane with draggable divider.

It can be either horizontal or vertical


### An example of 'un-styled' vertical DragResize

<Canvas>
  <Story name="basic-vertical">
    {() => {
      return (
        <DragResize
          firstPane={<div>left panel content</div>}
          secondPane={<div>right panel content</div>}
        />
      );
    }}
  </Story>
</Canvas>

### An example of 'un-styled' horizontal DragResize

<Canvas>
  <Story name="basic-horizontal">
