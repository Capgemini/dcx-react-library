# List

A List component ready to use in your form.  
List is UI/UX agnostic so you need to provide your style to have the look and feel you prefer.  
When you import the List component without providing any className or style associated will looks as following:

<Canvas of={BreadcrumbStories.Unstyled} />

A more complex example with all the possible properties is:

```js
<Breadcrumb
  className="myStyle"
  itemsClassName="myStyle"
  separatorItem={<span className="separator"> &gt; </span>}
  itemSelectedClassName="myStyle"
  breadcrumbsProps={{ style: { color: 'red' } }}
>
  <BreadcrumbItem
    selectedClassName="myStyle"
    breadcrumbItemProps={{ style: { color: 'red' } }}
    className="myStyle"
    selected={true}
  >
    content
  </BreadcrumbItem>
  <BreadcrumbItem5>content 2</BreadcrumbItem>
</Breadcrumb>
```
