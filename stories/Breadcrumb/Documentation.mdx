# Breadcrumb

A Breadcrumb component ready to use in your form.  
Breadcrumb is UI/UX agnostic so you need to provide your style to have the look and feel you prefer.  
When you import the Breadcrumb component without providing any className or style associated will looks as following:

<Canvas of={BreadcrumbStories.Unstyled} />

A more complex example with all the possible properties is:

```js
<Breadcrumb
  className="myStyle"
  itemsClassName="myStyle"
  separatorItem={<span className="separator"> &gt; </span>}
  itemSelectedClassName="myStyle"
  breadcrumbsProps={{ style: { color: 'red' } }}
>
  <BreadcrumbItem
    selectedClassName="myStyle"
    breadcrumbItemProps={{ style: { color: 'red' } }}
    className="myStyle"
    selected={true}
  >
    content 1
  </BreadcrumbItem>
  <BreadcrumbItem5>content 2</BreadcrumbItem>
</Breadcrumb>
```
