# GOVUK Style

In this section we're using the formInput component providing the GovUk style passing the relative `className`. Feel free to use your own css and style the formInput as you prefer

# Basic

<Canvas>
  <Story name="basic">
    {() => {
      const [value, setValue] = React.useState('');
      const handleChange = (event) => setValue(event.currentTarget.value);
      return (
        <FormInput
          name="text"
          type="text"
          inputProps={{ className: 'govuk-input' }}
          value={value}
          onChange={handleChange}
          ariaLabel="standard-input"
          ariaRequired = "true"
        />
      );
    }}
  </Story>
</Canvas>

# Dynamic error

Note: to display the error message type a letter and remove it

<Canvas>
  <Story name="error-message">
    {() => {
      const [value, setValue] = React.useState('');
      const handleChange = (event) => setValue(event.currentTarget.value);
      const handleValidity = (valid) => console.log(valid);
      return (
        <FormInput
          name="text3"
          type="text"
          value={value}
          onChange={handleChange}
          isValid={handleValidity}
          inputClassName="govuk-input"
          errorProps={{
            className: 'govuk-error-message',
          }}
          validation={{
            rule: {
              type: 'string',
              minLength: 1,
            },
            message: 'Enter an event name',
          }}
          errorMessage={{
            className: 'govuk-error-message',
          }}
          errorPosition="before-label"
          ariaLabel="standard-input-validation"
          ariaRequired = "true"
        />
      );
    }}
  </Story>
</Canvas>
