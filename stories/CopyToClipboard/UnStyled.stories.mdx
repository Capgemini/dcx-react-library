# CopyToClipboard

A CopyToClipboard component in case you want to copy the content of a specific elemene.
It supports:

- input
- textArea

and allow you to specifiy a custom content (whatever tag you prefer) or an image

CopyToClipboard is UI/UX agnostic so you need to provide your style to have the look
and feel you prefer.


### An example of 'un-styled' CopyToClipboard

<Canvas>
  <Story name="basic">
    {() => {
      const [value, setValue] = React.useState('');
      const inputRef = React.useRef(null);
      return (
        <>
         <input
            value={value}
            ref={inputRef}
            onChange={evt => setValue(evt.currentTarget.value)}
          />
          <CopyToClipboard
              ref={inputRef}
              onCopy={value => {
                alert(value);
              }}
          />
        </>
      );
    }}
  </Story>
</Canvas>
