# Input with Mask

An Input component ready to use in your form with support for input mask (eg. phone,date,number,etc).  
For the list of validation rules please refer to [imask](https://imask.js.org/)  
FormInputMasked is UI/UX agnostic so you need to provide your style to have the look and feel you prefer.
