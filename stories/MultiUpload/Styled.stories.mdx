# GOVUK Style

In this section we are using the MultiUpload component providing the GOV.UK Design System styles by passing the relative `className`. Feel free to use your own css and style the MultiUpload as you prefer

<Canvas>
  <Story name="styled">
    {() => {
      const handleChange = file => alert(`${file.name} was uploaded, it was last modified at ${new Date(file.lastModified).toLocaleDateString("en-us")}`);
      return (
        <MultiUpload
          name="file-upload-1"
          label="Upload a file"
          id="file-upload-1"
          className="govuk-form-group"
          inputProperties={{
            className: 'govuk-file-upload',
          }}
          labelProperties={{
            className: 'govuk-label',
          }}
          onChange={handleChange}
        />
      );
    }}
  </Story>
</Canvas>
