# Heading

A Heading component ready to use in your Typography
Heading is UI/UX agnostic so you need to provide your style to have the look and feel you prefer.  
When you import the Heading component without providing any className or style associated will looks as following:

<Canvas of={HeadingStories.Unstyled} />

A more complex example with all the possible properties is:

```js
<Heading
    label="This is a Heading"
    level="1"
    className="title"
    id="user-defined-id" 
/>
```
