# Button

A Button component ready to use in your form with support for delayClick and add an image before or after your label.

Button is UI/UX agnostic so you need to provide your style to have the look
and feel you prefer.

An example with all the available properties is:

## Usage

```js
<Button
  label="Start Now"
  onClick={buttonHandler}
  disableClickForMs={2000}
  disable={true}
  isLoading={true},
  loadingLabel="loading..."
  customLoadingPreImage={<span>spinner</span>}
  customLoadingPostImage={<span>spinner</span>}
  customPostfixImg={
    <svg
      xmlns="http://www.w3.org/2000/svg"
      width="17.5"
      height="19"
      viewBox="0 0 33 40"
      aria-hidden="true"
      focusable="false"
    >
      <path fill="currentColor" d="M0 0h13l20 20-20 20H0l20-20z"></path>
    </svg>
  }
  customPrefixImg={
    <svg
      xmlns="http://www.w3.org/2000/svg"
      width="17.5"
      height="19"
      viewBox="0 0 33 40"
      aria-hidden="true"
      focusable="false"
    >
      <path fill="currentColor" d="M0 0h13l20 20-20 20H0l20-20z"></path>
    </svg>
  }
  formAction="https://www.capgemini.com/gb-en/"
  name="submit"
  value="value"
/>
```
