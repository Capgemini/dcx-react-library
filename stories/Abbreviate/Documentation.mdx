# Abbreviate

A abbreviate component ready to use in your Typography.  
Abbreviate is UI/UX agnostic so you need to provide your style to have the look and feel you prefer.  
When you import the abbreviate component without providing any className or style associated will looks as following:  

<Canvas of={AbbreviateStories.Unstyled} />

A more complex example with all the possible properties is:

```js
<Abbreviate
  title="Laugh Out Loud"
  value="LOL"
  className="my-classname"
/>
```
