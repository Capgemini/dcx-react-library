# Select

FormSelect component ready to use in your UI

FormSelect is UI/UX agnostic so you need to provide your style to have the look
and feel you prefer.

## Usage

Use the FormSelect component when users need to select a value form a list

An example with all the available properties is:

```js
const [value, setValue] = React.useState('');
return (
  <FormSelect
    options={[
      { label: 'option1', value: 'value1' },
      { label: 'option2', value: 'value2' },
    ]}
    value={value}
    onChange={evt => setValue(evt.currentTarget.value)}
  />
);
```
