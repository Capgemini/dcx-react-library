# FormCheckbox

A checkbox component ready to use in your form.

FormCheckbox is UI/UX agnostic so you need to provide your style to have the look and feel you prefer.

## Usage

Use the checkbox component when users can only select one option from a list.

Conditional Rendering - When the conditional rendering object is defined, it will be displayed when the user selects the checkbox.
However if the user has javascript disabled (_progressive enhancement_) then the conditional content will _always_ be displayed.

An example with all the available properties is:

```js
<FormCheckbox
  name="checkbox-name"
  label="checkbox-label"
  value={value}
  onChange={handleChange}
  id="checkbox-id"
  ariaLabel="aria-label"
  ariaDescribedBy="aria-described-by"
  ariaLabelledBy="aria-labelledby"
  ariaDataControls="data-aria-controls"
  hint={{
    text: 'hint text',
    className: 'class names',
    id: 'hint-id',
    position: 'above',
  }}
  hintPosition="above"
  inputProps={{
    className: 'class-name',
  }}
  itemProps={{
    className: 'class-name',
  }}
  labelProps={{
    style: { display: 'flex' },
  }}
  disabled={true}
  defaultChecked={checked}
  conditional={{
    name: 'conditional-reveal',
    label: 'label',
    type: 'text',
    className: 'classes',
    groupClassName: 'group-classes',
    id: 'conditional-id',
    inputClassName: 'input-classes',
    inputId: 'input-id',
    labelClassName: 'label-classes',
    value: 'my-value',
    position: '',
  }}
  nested={false}
  inputClassName="inputClassName"
  labelClassName="labelClassName"
  itemClassName="itemClassName"
/>
```
